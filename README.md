# Serious Stratagem Hell — Python / Pygame 🐍

Pygame port of the *Helldivers 2*-inspired stratagem memory game.

> Also available in: [TypeScript/React](https://github.com/bourbask/SeriousStratagemHell) · [Rust](https://github.com/bourbask/SeriousStratagemHell/tree/rust) · [Go](https://github.com/bourbask/SeriousStratagemHell/tree/go)

## Quick start

```bash
pip install -r requirements.txt
python main.py
```

## Controls

- Directional arrows or **Z** (up), **Q** (left), **S** (down), **D** (right)
