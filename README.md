# Serious Stratagem Hell 🎮

Multi-language implementation of a **Stratagem Hero** arcade game inspired by *Helldivers 2*.

Each branch contains a full independent implementation:

| Language | Branch | Status |
|----------|--------|--------|
| TypeScript / React (web) | `main` | ✅ In progress |
| Python / Pygame | [`python`](https://github.com/bourbask/SeriousStratagemHell/tree/python) | ✅ Ported from original |
| Rust | [`rust`](https://github.com/bourbask/SeriousStratagemHell/tree/rust) | 🚧 Planned |
| Go | [`go`](https://github.com/bourbask/SeriousStratagemHell/tree/go) | 🚧 Planned |

---

## Web version (this branch)

React + TypeScript + Vite frontend with an Express API backend.

### Quick start

```bash
bun install --cwd backend
bun install --cwd frontend
bun run --cwd backend dev &
bun run --cwd frontend dev
```
